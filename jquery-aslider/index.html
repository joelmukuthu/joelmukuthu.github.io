jquery-aslider

slider plugin for jQuery

features:

  • supports CSS transitions
  • supports grid (vertical, horizontal and diagonal) transitions
  • plays nice with responsive designs
  • supports items (pages) of unequal sizes
  • provides an API for you to programmatically initiate transitioning to an item
  • does not add unnecessary CSS to the elements
  • built in support for up, down, left, right, previous, next, first and last buttons
  • built in support for keyboard navigation
  • supports before and after-slide callbacks

to navigate the pages:

  • use the keyboard arrow keys
  • click on the pagination (top right)
